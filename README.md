  <h1>Flutter E-Commerce App</h1>

  <p>
    A Flutter-based e-commerce application built using clean architecture
    principles and modern state management.
  </p>

  <h2>Features</h2>
  <ul>
    <li>Product listing using DummyJSON API</li>
    <li>Product search functionality</li>
    <li>Add to cart and order placement</li>
    <li>Order summary and order history</li>
    <li>Session-based authentication</li>
  </ul>

  <h2>Technologies Used</h2>
  <ul>
    <li>Flutter & Dart</li>
    <li>BLoC for state management</li>
    <li>DummyJSON API for product data</li>
    <li>HTTP package for API calls</li>
    <li>Hive for local storage</li>
  </ul>

  <h2>How It Works</h2>
  <p>
    The app fetches product data from an external API, manages user sessions
    locally, and allows users to browse, search, add items to cart, and place
    orders. All state changes are handled using BLoC for predictable and
    scalable architecture.
  </p>

  <h2>Run Instructions</h2>
  <ul>
    <li>Clone the repository</li>
    <li>Run <code>flutter pub get</code></li>
    <li>Run <code>flutter run</code></li>
  </ul>
